# face_mask
Face Mask Detection is a Machine Learning project that automatically identifies whether a person is wearing a face mask or not using images or live video. The system uses computer vision and deep learning techniques to analyze facial features and classify them into Mask or No Mask categories.

Libarary Used 
sklearn used this for to train and test data and split according to the needs
numpy used it for the numeric calculation
tensorflow used this for Sequential ,Conv2D, MaxPooling2D, Flatten, Dense,to_categorical
